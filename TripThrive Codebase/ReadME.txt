TripThrive – Aplicație web pentru organizarea călătoriilor

Accesarea proiectului

Proiectul TripThrive poate fi accesat și utilizat în următoarele moduri:


1. Accesare direct online

Aplicația este disponibilă public și poate fi accesată direct din browser:

https://www.tripthrive.app


Este necesară doar o conexiune la internet. 


2. Rulare locală în Visual Studio Code

Pentru accesarea proiectului local:

Descărcați proiectul pe calculator

Deschideți folderul proiectului în Visual Studio Code

Deschideți fișierul index.html

Rulați aplicația folosind extensia Live Server
sau deschideți fișierul direct în browser

Aplicația nu necesită configurări suplimentare.



3. Accesare prin GitHub Repository

Codul sursă al aplicației este disponibil pe GitHub:

https://github.com/Radukutz1/TripThrive
